# YorkTest Rapor Çevirici

YorkTest gıda intolerans test raporlarını İngilizce'den Türkçe'ye otomatik çeviren web uygulaması.

## Kurulum Adımları

### 1. GitHub Hesabı Oluşturun
- https://github.com adresine gidin
- Sign up butonuna tıklayın
- E-posta, şifre belirleyin

### 2. Yeni Repository Oluşturun
- GitHub'da New repository butonuna tıklayın
- Repository adı: yorktest-ceviri
- Public seçin
- Create repository butonuna tıklayın

### 3. Dosyaları Yükleyin
Aşağıdaki 3 dosyayı repository'ye yükleyin:
- app.py
- requirements.txt
- Premium food&drink list_179 (1).xlsx

### 4. Streamlit Cloud'a Deploy Edin
- https://streamlit.io/cloud adresine gidin
- Sign up with GitHub ile giriş yapın
- New app butonuna tıklayın
- Repository'nizi seçin
- Main file: app.py
- Deploy butonuna tıklayın

### 5. Hazır!
Birkaç dakika sonra uygulamanız hazır olacak ve size bir link verilecek.

## Kullanım
1. Web uygulamasının linkine tıklayın
2. DOCX dosyasını yükleyin
3. Çevir butonuna basın
4. Türkçe DOCX'i indirin

## Teknik Detaylar
- Framework: Streamlit
- Dil: Python 3.9+
- Çeviri Listesi: 179 gıda maddesi
- Format: DOCX (Word)
